# Sistema de Control de Equipos (v5.0.0)

## Descripción
Sistema de gestión de inventario desarrollado en C. Permite el registro, consulta, modificación, eliminación de equipos y la generación de reportes en formato de texto. Incluye validación estricta de datos y prevención de duplicados.

## Requisitos
- Compilador GCC (o compatible)
- Sistema operativo Windows, Linux o macOS

## Compilación
```bash
gcc -o sistema_control_equipos sistema_control_equipos.c
